An interactive, production-grade Single-Page Application (SPA) tracking portfolio matrices, built to mimic an immutable Security Operations Center (SOC) shell environment. This system operates entirely under the **Separation of Concerns (SoC)** software architecture paradigm, blending cryptographic and data science engineering tracking tools with a futuristic visual command wrapper.

---

## 🎨 Visual Architecture & Design Spec

* **Theme Blueprint:** "Deep Amethyst & Sapphire" High-Density UI. Built over deep space-blacks (`#070a13`) to mirror advanced security monitoring platforms.
* **Canvas Processing Subsystem:** A matrix-style falling digital rain background engine built natively onto an HTML5 2D context tracking frame, looping non-blocking parameters at a clean, consistent framerate via `requestAnimationFrame`.
* **Adaptive Breakpoint Matrix:** Implements a mobile-first fluid layout using utility-first layout constraints. Smooth contextual scroll event tracking dynamically highlights active navigation anchors on the fly.

---

## 📊 Feature Deep-Dive

### 1. Hardened Mock Terminal Widget
An interactive secure shell architecture configured with client command routing listeners.
* **XSS Countermeasures:** User terminal buffer lines are sanitized through strict regular expression character mapping arrays (`escapeHtml`), mitigating local script injections and structural payload execution vulnerabilities.

### 2. Live Core Metrics Telemetry Dashboard
A dual-panel analytics command deck natively rendering key metrics inspired by enterprise auditing monitors.
* **GitHub Operations Tracking:** Simulates real-time commit loops, active PR counts, and continuous version activities across branches.
* **LeetCode Problem Analytics Tracker:** High-fidelity code challenge tracking layout mapped via interactive CSS progress bars detailing solution distribution boundaries (Easy, Medium, Hard) and technical acceptance ratios.
